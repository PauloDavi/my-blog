# My blog

* Feito em next JS
* Tailwind
* Utiliza Typescript
* Posts gerados a partir de markdown
* Simples de configurar
* Com um layout padrão

## Possiveis Melhorias

* Utilizar CMS para cadastro de posts
* Colocar animações
* Criar Tela de categorias
